# zjut-smart-reserve

## 快速启动

### 1) 启动后端

在 `backend` 目录运行：

```bash
sh mvnw spring-boot:run
```

默认端口：`8080`  
数据库配置：`backend/src/main/resources/application.yml`

### 2) 启动前端

在 `frontend` 目录运行：

```bash
npm install
npm run dev
```

默认端口：`5173`
